# System (apt-get) packages for Streamlit Community Cloud -- auto-detected
# and installed alongside requirements.txt. This is NOT for Python
# packages (those go in requirements.txt / requirements-app.txt); this is
# specifically for OS-level dependencies.
#
# These two font packages provide the metric-compatible open-source
# substitutes the app swaps in when you pick "Arial", "Helvetica", or
# "Times New Roman" as a font (see FONT_SUBSTITUTIONS / resolve_font_family
# in giwaxs_common.py) -- without them installed at the OS level, those
# options either won't appear in the picker, or would silently render as
# matplotlib's default font instead of the one you actually chose.
fonts-liberation
fonts-texgyre
